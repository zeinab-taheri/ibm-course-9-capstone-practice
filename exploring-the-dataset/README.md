# Exploring the Dataset

Clean practice version of **Lab 5: Exploring the Dataset** from the IBM Data Analyst Capstone.

## What this notebook covers
- loading a CSV dataset with `pandas`
- previewing the first rows with `head()`
- checking the number of rows and columns with `shape`
- reviewing column data types with `dtypes`
- estimating mean age by mapping age ranges to numeric values
- counting unique countries in the `Country` column
