# Web Scraping Basics

This folder contains a beginner web scraping practice notebook based on the IBM Data Analyst Capstone.

## What this practice covers
- Downloading webpage HTML with `requests`
- Parsing HTML with `BeautifulSoup`
- Extracting links from `<a>` tags
- Extracting image sources from `<img>` tags
- Scraping values from an HTML table

## Files
- `web_scraping_basics.ipynb` — cleaned practice notebook
- `requirements.txt` — basic dependencies
- `README.md` — short project explanation

## Tools used
- Python
- requests
- BeautifulSoup

## Notes
This is a basic review exercise for learning web scraping fundamentals. It is not an advanced or production scraping project.
