# Web Scraping Data Collection

This folder contains a beginner web scraping data collection notebook based on the IBM Data Analyst Capstone.

## What this practice covers
- Downloading webpage HTML with `requests`
- Parsing HTML with `BeautifulSoup`
- Finding an HTML table
- Extracting programming language names and salaries
- Cleaning salary text values
- Saving the results into a CSV file

## Files
- `web_scraping_data_collection.ipynb` — cleaned practice notebook
- `popular-languages.csv` — output file created from the scrape
- `requirements.txt` — basic dependencies
- `README.md` — short project explanation

## Tools used
- Python
- requests
- BeautifulSoup
- csv

## Notes
This is a basic web scraping data collection exercise for learning how to extract table data from a webpage and save it into a CSV file. It is not an advanced or production scraping project.
