# Web Scraping Data Collection

This folder contains a cleaned practice version of IBM Data Analyst Capstone Lab 4.

## What this practice covers
- Downloading webpage HTML with `requests`
- Parsing HTML with `BeautifulSoup`
- Finding an HTML table and its rows
- Extracting programming language names and salary values
- Cleaning salary text by removing `$` and `,`
- Saving the scraped results into a CSV file

## Files
- `web_scraping_data_collection.ipynb` — cleaned practice notebook
- `popular-languages.csv` — scraped output file
- `requirements.txt` — basic dependencies
- `README.md` — short project explanation

## Tools used
- Python
- requests
- BeautifulSoup
- csv
- pandas (only for optional output check)

## Notes
This is a beginner web scraping exercise focused on collecting structured table data and exporting it to CSV.
